작성 완료를 누르면 답변이 Markdown(.md) 문서로 저장됩니다.

Markdown은 메모장처럼 가볍지만 제목, 질문, 답변 구조가 깔끔하게 남는 문서 형식입니다. AI가 나중에 인터뷰 내용을 읽고 분석하기 좋아서, AX 컨설팅 자료는 이 형식으로 모읍니다.

작성 완료: 저장 및 발송을 누르면 먼저 내 PC에 저장할 `.md` 파일 위치를 선택합니다. 저장이 끝난 뒤 파일 첨부 공유창을 엽니다.

브라우저가 지원하면 Gmail이나 Outlook 선택창에 `.md` 파일이 함께 전달됩니다.

지원하지 않는 브라우저에서는 메일 작성 화면만 열립니다. 이 경우 저장된 `.md` 파일을 직접 첨부해주세요.

휴대폰에서는 파일 위치 선택 대신 `.md` 파일 다운로드로 저장될 수 있습니다.

메일은 바로 발송되지 않고 작성 화면으로 열립니다. 내용을 확인한 뒤 직접 보내면 됩니다.
